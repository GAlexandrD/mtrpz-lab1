# Quadratic Equation Solver 
Application to solve quadratic equations

## Setup

install: 
```bash
npm install
```
run app: 
```bash
npm start <filepath>
```
filepath is optional

## File format

```
A\sB\sC(\n | \r\n)
```
A, B, C - real numbers, A != 0.
